Documentation

Model Context Protocol (MCP)

MCP is like USB-C for AI: it builds a secure, two-way connection between any external data source and an AI/LLM system. It enables structured, seamless context-sharing for smart, dynamic workflows.

Google Agent Development Kit (ADK)

Google ADK provides tools for building AI agents with Gemini models, enabling complex workflows and agent-based problem solving.

Integration Architecture

The MCP-ADK integration consists of two main components:

  • A TypeScript MCP service that handles context management
  • A Python ADK service that provides agent-based workflows

Getting Started

To use the integration, follow these steps:

  1. Set up your environment with the required credentials
  2. Run both services using Docker or locally
  3. Use the API endpoints to make requests

API Reference

Playground Endpoint

POST /api/playground

This endpoint allows you to send research topics to the integration.

Request Body

{
  "topic": "Your research topic",
  "use_adk": true|false
}

Response

// When use_adk is true
{
  "report": "Generated report from ADK"
}

// When use_adk is false
{
  "result": "Standard MCP response",
  "topic": "Your research topic",
  "timestamp": "ISO timestamp"
}

Generate Report Endpoint

POST /generate_report

This endpoint is used by the ADK service directly.

Request Body

{
  "topic": "Your research topic"
}

Response

{
  "report": "Generated report about the topic",
  "status": "success"
}

Examples

Example 1: Basic Topic Research

curl -X POST http://localhost:5000/api/playground \
-H "Content-Type: application/json" \
-d '{"topic": "Future of AI", "use_adk": true}'

Example 2: Standard MCP Response

curl -X POST http://localhost:5000/api/playground \
-H "Content-Type: application/json" \
-d '{"topic": "Climate Change", "use_adk": false}'

Example 3: Direct ADK Request

curl -X POST http://localhost:8000/generate_report \
-H "Content-Type: application/json" \
-d '{"topic": "Sustainable Energy"}'

Example 4: Python Client

import requests

response = requests.post(
    "http://localhost:5000/api/playground",
    json={"topic": "Machine Learning", "use_adk": True}
)

print(response.json())
